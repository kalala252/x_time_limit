# X Direct Access Blocker (Chrome)

`x.com` / `twitter.com` の直接アクセスを 1 日 15 分まで許可し、それを超えるとブロックする Chrome 拡張です。  
他サイト内の埋め込み表示（iframe）は許可されます。

## 使い方

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. この `chrome-x-blocker` フォルダを選択
5. 既に読み込み済みの場合は、拡張カードの「再読み込み」を押す

## 動作

- 1 日あたりの直接閲覧上限: `15 分`
- カウント対象:
  - `x.com` / `twitter.com` をトップレベルで開いている
  - そのタブがアクティブ
  - ブラウザウィンドウがフォーカスされている
- 上限到達後:
  - 直接アクセスは `blocked.html` に遷移
- 常時許可:
  - 他サイトに埋め込まれた X/Twitter 表示（iframe）

## 補足

- 拡張アイコンのバッジに残り分数（例: `18`）を表示します。
- バッジ色:
  - 緑: 今まさに計測中（X/Twitter のアクティブ表示中）
  - 赤: 非計測中（他サイト表示中 / 非アクティブ / 上限到達）
- 毎日ローカル時刻の午前 5:00 を境に使用時間が自動リセットされます。
- 実装は `background.js` で計測し、`block.js` で上限時のみブロック判定します。
- 反映確認はシークレットウィンドウ（拡張を許可した状態）で行うとキャッシュ影響を避けやすいです。
